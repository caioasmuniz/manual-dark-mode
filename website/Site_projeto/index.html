Manual Dark Mode

Devido ao hábito de muitos usuários utilizarem seus dispositivos em ambientes de pouca iluminação, onde o brilho da tela acaba incomodando os usuários, o modo escuro (Dark Mode) começou a se popularizar cada vez mais nos últimos anos. Porém, embora muitos aplicativos recentes possuam um modo escuro como opção, isto não é padronizado para páginas web, sendo assim, criamos essa plataforma onde os usuários podem inserir arquivos CSS para aplicar modo escuro em páginas que não ofereçam suporte ou onde as extensões existentes apresentem erros que impossibilitem seu uso.
